# API

The NestJS API is versioned under `/v1` and exposes Swagger at `/docs`.

## Initial endpoints

- `GET /v1/health`
- `POST /v1/auth/register`
- `POST /v1/auth/login`
- `POST /v1/flights/search`
- `POST /v1/bookings`
- `POST /v1/payments/intent`
- `POST /v1/payments/webhook`
- `POST /v1/ai/chat`
