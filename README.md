# Nexaris Travel

A polished, responsive travel landing page for Nexaris Travel. The project is intentionally dependency-free so it can run in restricted environments while still providing a production-ready static site.

## Scripts

- `npm run dev` — start the local static server from `public/`.
- `npm run build` — copy the app into `dist/` for deployment.
- `npm run lint` — run lightweight repository checks.
- `npm run preview` — serve the production build from `dist/`.
